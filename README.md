# Jaguar TV Partner Platform MVP

Mobile-first recruitment platform for Brazil-based Jaguar TV partners during the World Cup window.

## Run locally

```bash
npm install
npm run dev
```

## Content updates

Most public copy, tutorial entries, material packs, navigation, and WhatsApp settings live in:

```text
data/site.ts
```

Replace `whatsappNumber`, `telegramUrl`, YouTube embed URLs, and material descriptions before launch.
